---
name: Lesson Planner
description: Day-wise sequential lesson plans (Day 1, Day 2...) embedded inside chapter accordions on ClassesPage, no dates/periods
type: feature
---
- Day plans are sequential per chapter (Day 1, 2, 3...), NOT date or period based
- Integrated directly into ClassesPage chapter accordion via DayPlanSection component
- Minimal teacher input: title + optional notes + optional file/camera upload
- Teachers can mark days as completed with checkbox toggle
- Chapters sorted alphabetically ascending by name (natural sort), no auto-generated serial index shown
- LessonPlannerPage is a summary dashboard linking to ClassesPage for actual management
