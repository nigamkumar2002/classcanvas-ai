---
name: Exam System
description: AI/manual/document exam creation, scheduling (instant/scheduled with date+time window), class-filtered, timed MCQ, auto-grading, analytics
type: feature
---
- Three creation modes: Manual, AI (topic-based), AI (from document/PDF)
- Exams linked to Class → Subject → Chapter hierarchy, optional day plan link
- Publish modes: Instant publish or Scheduled (date + optional start/end time window)
- Students locked out before/after scheduled window
- Super admin/admin can permanently delete exams (type "DELETE" confirmation)
- Leaderboard visibility toggle per exam (teacher decides)
- Analytics tab: Student view (history, ranking, question breakdown, trends), Teacher/Admin view (pass rates, weak students, leaderboard, per-student drill-down)
- Cascade delete: removing exam removes questions + results automatically
- Content approval workflow: teacher exams need admin approval before becoming active
